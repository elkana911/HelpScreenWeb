  3. Untuk konsumen yang melakukan pembayaran input di menu Payment Receive.

  4. Untuk input hasil kunjungan (tidak ada pembayaran) di menu Visit Result Entry.
    Input dengan lengkap semua isian yang tersedia.

  5. Untuk Input penarikan kendaraan di menu Repo Entry.
    Pastikan nomor BASTBJ sesuai dengan fisik.

  6. Setelah selesai melakukan proses penagihan dan kunjungan, sebelum melakukan setor uang ke kasir, kolektor harus melakukan Close Batch.
